# Проверка работы (преподаватель)

## Быстрая проверка компиляции

Из корня репозитория (каталог с `README.md`):

```bash
find part* company/core -name '*.java' -print0 | xargs -0 javac --release 21 -encoding UTF-8
```

Файл `company/app/HRSystem.java` **не включён**: он должен оставаться с ошибками до выполнения задания 1.2.

После того как студент исправит задание 1.2, можно проверить весь пакет `company`:

```bash
javac --release 21 -encoding UTF-8 company/core/*.java company/app/*.java
```

Ожидание: без ошибок компилятора.

## Сравнение с эталонным репозиторием

1. Добавить `upstream`: `git remote add upstream git@github.com:Mohanad0101/practice2.git` (если ещё не добавлен).
2. `git fetch upstream`
3. Просмотр отличий: `git diff upstream/main --stat` или покоммитно по дате сдачи.
