# Слайды для доклада (ООП и практика 2)

## Как открыть

1. Откройте файл **`index.html`** двойным щелчком или перетащите в браузер (Chrome / Firefox / Edge).
2. Либо в IntelliJ: правый клик по `index.html` → **Open in** → Browser.

## Управление

| Действие | Клавиша / элемент |
|----------|-------------------|
| Следующий слайд | `→`, `Пробел`, `Page Down`, кнопка «Вперёд» |
| Предыдущий слайд | `←`, `Page Up`, кнопка «Назад» |
| Первый / последний | `Home` / `End` |
| Заметки докладчика | Кнопка «Заметки докладчика» или клавиша **`N`** |

Панель **«Заметки докладчика»** — краткие подсказки, что сказать устно; студентам можно показывать только основной экран.

## Содержание (10 слайдов)

1. Титул и три столпа ООП  
2. Инкапсуляция + связь с лаб. (1.1, 1.2)  
3. Наследование + abstract (2.1)  
4. Полиморфизм (2.1, 2.2, 8.1)  
5. Класс / abstract / interface / sealed  
6. record / enum / аннотации  
7. Паттерны: шаблонный метод, стратегия, фасад, Stream  
8. Массивы и строки (ч. 3–4)  
9. Лямбды и Stream (ч. 7)  
10. Карта частей репозитория  

Файл **автономный** (шрифты подгружаются с Google Fonts; нужен интернет при первом открытии).
